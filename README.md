## üìù Project Description
This is a comprehensive school management system for Lord Buddha School, built with PHP, MySQL, and modern CSS. The system includes:

- **Public Website**: Showcases school information, academics, admissions, campus life, and notices.
- **Admin Portal**: Allows administrators to manage students, teachers, classes, subjects, exams, exam marks, and announcements.
- **Teacher Portal**: Enables teachers to view attendance, manage marks, and access relevant information.
- **Student Portal**: Provides students with access to their marks, notices, and personal dashboard.

The project emphasizes a user-friendly interface with role-based access control, ensuring secure and efficient management of school operations.

This is a local development project. To run:

1. Set up a local server (e.g., XAMPP, WAMP) with PHP and MySQL.
2. Create a database named `schoolms`.
3. Import the `sample_data.sql` file to populate the database with sample data.
4. Update database credentials in `config/db.php` if needed.
5. Access via `http://localhost/schoolms/public/index.php`
